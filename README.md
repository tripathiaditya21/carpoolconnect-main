# 🚗 CarpoolConnect

CarpoolConnect is a smart ride-sharing platform designed to help users **find, share, and manage carpool rides efficiently**. It aims to reduce travel costs, traffic congestion, and carbon emissions by connecting people traveling in the same direction.

---

## 🌟 Features

* 🔍 **Find Rides** – Search for available carpools based on location, time, and destination
* 🚘 **Offer Rides** – Drivers can publish ride details and accept passengers
* 👥 **User Authentication** – Secure login/signup system
* 📍 **Real-time Location Matching** – Match users based on routes
* 💬 **Communication System** – Chat or contact between drivers and riders
* ⭐ **Ratings & Reviews** – Build trust within the community
* 📱 **Responsive Design** – Works across devices

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/CarpoolConnect.git
cd CarpoolConnect
```

### 2️⃣ Install dependencies

**Frontend:**

```bash
cd client
npm install
```

**Backend:**

```bash
cd server
npm install
```
### 4️⃣ Run the application

**Backend:**

```bash
npm start
```

**Frontend:**

```bash
npm start
```
